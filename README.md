## 1. このプロジェクトは何か

Niu株式会社のコーポレートサイト（1ページ完結のホームページ）。
**`index.html` 1枚だけで動く静的サイト**です。フレームワークやビルド作業は不要。

- 内容: 会社紹介 ＋ サービス（SNS運営 / 中国語顧問 / 中国市場開拓戦略(RED・小紅書) / AI開発 / 言語学習アプリ開発）。不動産事業は「準備中」として掲載（宅建免許 未取得のため宅建業の表現・営業表示はしていない）。料金（プラン）は非掲載

- デザイン: ロゴ（紫の蝶）に合わせた、パープル × クリーム色のかわいい雰囲気
- 蝶々のアニメーション（飛び交う蝶・羽ばたき・スクロール表示など）入り

## 2. 新しいデバイスで始める手順（クイックスタート）

### A. とにかく見るだけ（最速・道具不要）
1. このフォルダ（`companiu`）一式を新しいPCにコピー
2. `index.html` を**ダブルクリック**してブラウザで開く
   → これだけで完成形が見られます（ロゴ画像も同じフォルダにあるので一緒に表示されます）

### B. 編集・プレビューしながら開発する
1. フォルダ一式をコピー
2. 下記「3. 必要な道具」をインストール
3. ターミナルでフォルダに移動してプレビューサーバーを起動（「5. プレビューの起動」参照）
4. `index.html` をエディタで編集 → ブラウザを再読み込みして確認

> 💡 **移行は「このフォルダごとコピーするだけ」**でOK。USBメモリ・クラウドドライブ（iCloud/Google Drive等）・GitHubのいずれでも可。

## 5. プレビューの起動（ローカルサーバー）

`index.html` はダブルクリックでも見られますが、開発中はローカルサーバー経由が確実です。

```bash
# フォルダに移動してから
python3 -m http.server 8755
```
→ ブラウザで <http://localhost:8755> を開く（停止は `Ctrl + C`）

> ポート 8755 は `.claude/launch.json` と合わせています。変えるときは両方を直してください。
> 編集後にブラウザに反映されないときは **強制リロード（Mac: `Cmd + Shift + R` / Win: `Ctrl + F5`）**。

## 7. ロゴ画像の更新手順

新しいロゴ（白 or 単色背景の画像）に差し替えるとき:

```bash
# 入力画像 → 背景透過＆余白トリミング → logo.png を生成
python3 tools/cutout_logo.py 新しいロゴ.png logo.png
```
- `index.html` は `logo.png` を参照しているので、上書きすれば自動で反映されます。
- うまく抜けない場合は `tools/cutout_logo.py` 内の `THRESHOLD` を調整（白背景向けの設定）。
- 元画像は念のため別名で残しておくと安心です。
